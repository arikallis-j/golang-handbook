# Типы данных в Go

1. Базовые:
   - Строки
   - Целые числа
   - Числа с плавающей точкой
   - Булевы значения
2. Композитные
   - Массивы и слайсы
   - Мапы
   - Структуры
3. Указатели

## Целые числа

- знаковые: `int`, `int8`, `int16`, `int32`, `int64`;
- беззнаковые: `uint`, `uint8`, `uint16`, `uint32`, `uint64`.
  
Если размерности `int64` недостаточно, можно воспользоваться типом `big.Int` из пакета `math/big`.

## Числа с плавающей точкой

- комплексные: `complex64`, `complex128`;
- вещественные: `float32`, `float64`.

## Базовые числовые операции

Для числовых типов доступны следующие арифметические операции:

- `+` — сложение,
- `-` — вычитание,
- `*` — умножение,
- `/` — деление нацело,
- `%` — остаток от деления.

Сокращенные операции:

- `a + b` -> `a += b`
- `a = a + 1` -> `a++`
- `a = a - 1` -> `a--`

## Булевы значения

- тип `bool`: `true`, `false`

Операции с булевыми значениями:

- `&&` — И
- `||` — ИЛИ
- `!` — НЕ

## Строки

- тип `string`: "любые тексты"

Строки в Go представляют собой массив из значений типа `byte`. Строка хранится как массив байт, но он неизменяемый.

1. Для ASCII символов длина символа 1 байт, поэтому функция `len` будет возвращать количество символов
2. Для Unicode символов длина больше 1 байта, поэтому брать длину строки в байтах некорректно
3. Для работы с Unicode есть тип `rune` и биюлиотека `unicode/utf8`
4. Задание строк можно сделать также через ``, что позволяет делать переносы и табы
5. Знак `+` поддерживает конкатенацию двух строк

## Пользовательские типы

- Пользовательские типы можно писать явно с помощью ключевого слова `type`.
- Для них также можно определять методы. Определять методы для встроенных типов в Go нельзя.
- Приведение типов происходит через функцию `<type>(variable)`

## Псевдонимы (aliases)

Псевдонимы позволяют обращаться к типу в коде по другому имени. Так, типы `rune` и `byte` представляют собой алиасы к `int32` и `uint8`:

```go
type rune = int32
type byte = uint8
```

## Значения по умолчанию

Все типы имеют значения по умолчанию, которыми автоматически инициируется объявленная переменная, если это не было сделано явно:

- для `bool` значение по умолчанию — `false`;
- для числовых типов — 0;
- для ссылочных типов — `nil` или пустой указатель;
- для `string` — пустая строка длиной 0.
