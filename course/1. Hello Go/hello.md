# Основные команды компилятора

- Скомпилировать код и получить бинарник:
  
```bash
go bulid test.go
```

- Сразу же и скомпилировать, и запустить код:
  
```bash
go run test.go
```

- Запустить кодогенерацию:

```bash
go generate test.go
```

- Cкачать утилиту, скомпилировать и установить бинарник в $GOPATH/bin 

```bash
go install github.com/...
```

- Скачать зависимости
  
```bash
go get 
```

- Создать файл со списком зависимостей
  
```bash
go mod init
```

- Актуализировать файл со списком зависимостей

```bash
go mod tidy
```

- Запустить unit-тесты/бенчмарки

```bash
go test
```

- Прогнать автоформатирование кода

```bash
go fmt test.go
```

- Вывести переменные окружения go
  
```bash
go env
```

- Вывести справку о команде
  
```bash
go help <command>
```
